# AI Travel Chatbot 🌍✈️

An AI-powered travel chatbot built using Flask and Groq AI. The chatbot helps users with travel-related questions, destination suggestions, and travel guidance through a simple web interface.

## Features

* AI-powered travel assistance
* Interactive chatbot interface
* Flask-based backend
* Fast and responsive web application
* Easy to customize and extend

## Technologies Used

* Python
* Flask
* Groq API
* HTML
* CSS
* JavaScript
